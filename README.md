# DS-Manuscript
## 前言？
喜欢`NiklausWirth`的一句话:**程序=数据结构+算法**。

但这篇文章的目的不在于去抠很多的理论内容，而是直截了当的通过画图结合的方式展现数据结构的基础操作以及代码演示，所以，这篇文章并**不适合用来作为应考**的小册子，更多的是学习数据结构的**实战与基本原理**。
